    Nike+ (2007)

    This project was for an advertisement for "Nike+iPod". The Nike+ running shoes have sensors that send information to the iPod. When someone runs or walks, the iPod picks up the signal and records the distance and pace of the workout. This off-line advertisement installation for "Nike+iPod" consists of a treadmill and a jumbo iPod. People try to run on the treadmill with Nike+ running shoes while checking the pace and distance of their workout on the jumbo iPod. I tried to make a runner on a treadmill feel like he was running on different surfaces such as gravel or sand on a beach by dubbing the sound of these places. I developed a module for receiving sensor signals and generating corresponding sounds with C language (the module was excluded in the final version because of marketing effectiveness of the exhibition).

    Wave Pong Pong (2006)

    This project was for a set of interactive mobiles for children. When one of mobiles detected children's movements, it started to react and spread the reaction to the other mobiles. The mobiles danced, turned lights on and off, and made sounds. I was part of a team that built the mobiles. I programmed PIC (Programmable Interface Controller) with PIC C language. The mobiles were exhibited at Goyang Children's Musium.

    Aurora (2006)

    This project is an interactive display system which consists of arrays of LEDs and sensors. Whenever it detects people under the system, it changes lighting patterns and the light is reflected in mirrors. The images look like an aurora. I participated in PIC programming for the system, which was installed at the Multi-functional Administrative City Information Center. There are hundreds of LEDs in the system. I used PIC programming to synchronize the lights so that I could control all of the different colors.

    Music Doorbell (2006)

    This system recognizes people's face and detects their feelings. The program creates a doorbell sound that reflects their emotions. This is a proof-of-concept system, which really doesn't work well. I used SVM (Support Vector Machine) to train the system to recognize the relationship between moods and their facial expressions. Then, I created a composing function for generating music based on the people's moods.
